# 使用方法：
输入完整压缩包路径（Windows从盘符开始输入，Android从根目录开始输入，~~你先别急，可视化文件选取在写的~~）<br>
压缩包必须包含config.json<br>
参数：<br>
Song：歌曲名称（包含扩展名），暂时只能是wav格式<br>
Chart：谱面文件名称（包含扩展名），暂时只能使用Phigros Official Chart Version 3 ~~（说实话我不太想去兼容远古Version1）~~<br>
